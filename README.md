# Experimental Comparison of Sorting Algorithms

This repository contains the implementation and experimental results for a sorting algorithms comparison project developed in C.

The implemented algorithms were:
- Bubble Sort
- Insertion Sort
- Selection Sort
- Pancake Sort
- Heap Sort
- Merge Sort
- Quick Sort
- Shell Sort

The project contains:
- C source files and header files
- Code::Blocks project file
- cleaned result files

The idea behind the experiment was:
The project compares the practical time efficiency of several sorting algorithms on inputs of different sizes and structures.  
The algorithms were separated into two categories: simple sorts and efficient sorts. This was done because the two groups do not scale similarly for large inputs.
